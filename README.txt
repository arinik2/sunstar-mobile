------------------------------------------------------------------------------------------
Пакеты, настройка
------------------------------------------------------------------------------------------
Чтобы работало QAudio после установки QtCretor (based on Qt 5.*, check qtcreator -version) 
в debian-подобных системах надо доставить следующие пакеты или удостовериться, что они есть:

sudo apt-get install libqtmultimediakit1
sudo apt-get install libqt5multimedia5-plugins
-----------------------------------------------------------------------------------------
Как тестить
-----------------------------------------------------------------------------------------
mplayer -rawaudio samplesize=2:channels=1:rate=44100 -demuxer rawaudio test.raw
В общем, это работает, но по ходу дела в любой стандартной записи есть еще и шумоподавление, а без него качество конечно не очень(
samplesize=2 - размер инфы о громкости в байтах, в то время как в проге этот же паоаметр указан в битах
rate - типа 44.1kHz
остальные параметры - без них чет не работает)
